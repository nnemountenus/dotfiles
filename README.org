#+TITLE: Инструментарий
* Emacs
  Основной текстовый/кодовый редактор

  - IDE-подобная среда для разработки на C, Lua (благодаря LSP)
  - Поддержка Git
  - Работа с файлами через SSH
  - Заметки, TODO-листы, календари с org-mode
  - Файловый менеджер dired, может в двухпанельный режим
  - Встроенный терминал
  - Читалка для PDF-файлов
* Firefox
  Ocновной браузер

  Привычный браузер с минимизированным интерфейсом.
